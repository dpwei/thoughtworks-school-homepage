# 思特沃克学院-首页

Features following:

- Webpack
- ES6
- Bootstrap v4
- Font Awesome
- Sass
- Live reload using Browsersync

## Getting Started

Run script to start devlopment
```sh
npm run start
```

For distribution folder run the following command and it will create a dist folder which can be placed on a server
```sh
npm run build
```
